## Para Acólitos

Este manual foi criado para auxiliar acólitos no desempenho de suas funções litúrgicas, especialmente no manuseio correto do Missal Romano. O serviço do altar é uma vocação importante que contribui para a dignidade das celebrações eucarísticas.

## Sobre o Missal Romano

O Missal Romano é o livro litúrgico oficial da Igreja Católica que contém todas as orações, leituras e instruções necessárias para a celebração da Santa Missa. Conhecer sua estrutura e uso adequado é fundamental para todo acólito.
